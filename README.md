# hbxss
Blind XSS Testing toolkit

### Install
git clone https://github.com/hahwul/hbxss.git<br>
cd hbxss<br>
bundle install<br>

### Run

rails s<br>
Open browser http://127.0.0.1:3000

### Usage
1. Generate code.<br>
2. Inject XSS Code your testing(target) site.<br>
3. Check the Blind XSS results on the log page.<br>
